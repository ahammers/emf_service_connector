a# EMF Service Connector (Home Assistant)



Sendet Energiewerte alle 5 Minuten an die EMF "Energiewende mit Freunden" API.



\## Installation (HACS)

\- Custom repository hinzufügen

\- Integration installieren

\- Home Assistant neu starten

\- Integration hinzufügen: "EMF Service Connector"



\## Konfiguration

Standard:

\- base\_url

\- api\_key (Entity)

\- site\_fid (Entity)

\- datapoint\_ts: now oder Entity

\- em\_power\_grid (Entity)



Erweitert:

\- optional weitere Felder (Entities)
